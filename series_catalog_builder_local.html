Générateur local de series_catalog.json

Important : si le serveur Xtream bloque le CORS, le navigateur empêchera l’enrichissement. Dans ce cas, le problème vient du serveur, pas de cette page.

1. Charger le fichier brut

Le fichier attendu ressemble à ton series.json actuel avec des URLs get_series_info.

2. Lancer l’enrichissement

Champ numérique = pause entre deux requêtes, en millisecondes. Monte à 300–500 ms si le serveur réagit mal.
